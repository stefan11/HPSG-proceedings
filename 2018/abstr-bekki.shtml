Bekki, Daisuke: lightblue: A Japanese CCG Parser

In this talk I address problems that arise from the use of category variables vT in combinatory categorial grammars (CCGs), in particular, that they require an extension of semantic languages with variable vectors. As a solution to such problems, we introduce a technique for implementing variable vectors within the context of lightblue, a Japanese CCG parser implemented within the Haskell programming language with a dependent type semantics (DTS) representation.
